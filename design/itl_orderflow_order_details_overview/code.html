description Описание проекта

Цель проекта — полное обновление пользовательского интерфейса внутреннего портала компании ТехноСофт. Основной упор делается на улучшение UX для мобильных сотрудников и интеграцию новых модулей отчетности.

  • Разработка адаптивной дизайн-системы.
  • Миграция данных со старой платформы (PHP 5.6) на современный стек.
  • Внедрение сквозной авторизации через корпоративный AD.
